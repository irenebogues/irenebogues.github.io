# Personal Website

This is a simple personal website built with Python.

## How to Build

Run:

```bash
python3 build_site.py
```

That creates:

- `index.html`
- `styles.css`

## How to Preview

Run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## How to Edit

Open `build_site.py` and change the beginner-friendly variables near the top:

- `NAME`
- `TITLE`
- `INTRO`
- `GITHUB_URL`
- `INSTAGRAM_URL`
- `LINKEDIN_URL`
- `CALENDLY_URL`
- `ABOUT`
- `PROJECTS`
- `LEARNING_GOALS`
- `PHOTOS`

Then run `python3 build_site.py` again.
